# SQLverse

**SQLverse** is an interactive platform designed to teach and enhance your SQL and DBMS skills. It offers comprehensive tutorials, a command reference, a case study system, and an SQL playground where users can practice and execute queries on real-world and custom datasets.

## Features

- **Interactive Tutorials**: Learn SQL concepts with detailed, static tutorials that include explanations and usage examples for common SQL commands.
- **SQL Command Reference**: A comprehensive reference for SQL commands, providing syntax, use cases, and practical examples.
- **Case Study System**: Pre-configured case studies with sample data for hands-on learning. Users can also create and execute queries against custom datasets.
- **SQL Playground**: Run SQL queries on default or custom datasets uploaded via Excel, JSON, or manual entry. This feature allows for experimentation and learning by doing.
- **User Authentication**: Secure login for personalized features like saving custom case studies and playground data.
- **Flashcards & Quizzes (Coming Soon)**: Interactive flashcards and quizzes to test and reinforce your knowledge.
- **AI Integration**: Future updates will include AI-powered analysis and optimization suggestions for your SQL queries.
